Generic Classes in Java

A Generic Class allows a class to work with different data types using type parameters.

Generics provide type safety and reduce the need for type casting.

Syntax

class ClassName<T> {
  T data;
}

Example

class Box<T> {
  T value;

  void set(T value) {
    this.value = value;
  }

  T get() {
    return value;
  }
}

Advantages

✔ Type safety
✔ Code reusability
✔ Compile‑time checking
