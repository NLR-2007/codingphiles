Variables in Java

Variables are containers used to store data values in Java.

Types of Variables

Java has three types of variables:
✔ Local Variables
✔ Instance Variables
✔ Static Variables

Local Variables

Declared inside methods and accessible only within the method.

Instance Variables

Declared inside a class but outside methods.

Static Variables

Declared using the static keyword and shared among all objects.

Variables (Exam Ready)

Variables store data values used during program execution.
