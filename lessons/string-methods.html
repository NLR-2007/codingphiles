String Methods in Java

The String class in Java provides a rich set of built‑in methods to perform operations on strings.

These methods help in:
✔ Manipulating text
✔ Comparing strings
✔ Searching content
✔ Formatting output

1. length()

Returns the number of characters in the string.

Example:

String s = "Java";
System.out.println(s.length());

Output:
4

2. charAt()

Returns the character at a specified index.

Example:

String s = "Java";
System.out.println(s.charAt(1));

Output:
a

3. toUpperCase()

Converts all characters to uppercase.

Example:

String s = "java";
System.out.println(s.toUpperCase());

Output:
JAVA

4. toLowerCase()

Converts all characters to lowercase.

Example:

String s = "JAVA";
System.out.println(s.toLowerCase());

Output:
java

5. trim()

Removes leading and trailing spaces.

Example:

String s = " Java ";
System.out.println(s.trim());

6. equals()

Compares the content of two strings.

Example:

String a = "Java";
String b = "Java";
System.out.println(a.equals(b));

Output:
true

7. equalsIgnoreCase()

Compares strings ignoring case differences.

Example:

String a = "Java";
String b = "java";
System.out.println(a.equalsIgnoreCase(b));

8. compareTo()

Compares two strings lexicographically.

Example:

"Java".compareTo("Java"); // 0
"Java".compareTo("Python"); // negative
"Python".compareTo("Java"); // positive

9. substring()

Returns a part of the string.

Example:

String s = "JavaProgramming";
System.out.println(s.substring(4));

Output:
Programming

10. replace()

Replaces characters or substrings.

Example:

String s = "Java";
System.out.println(s.replace("a", "o"));

Output:
Jovo

11. contains()

Checks whether a string contains a specific sequence of characters.

Example:

String s = "Java Programming";
System.out.println(s.contains("Java"));

12. startsWith() and endsWith()

Checks starting or ending characters.

Example:

"Java".startsWith("Ja"); // true
"Java".endsWith("va"); // true

13. split()

Splits a string into an array based on a delimiter.

Example:

String s = "Java,Python,C";
String[] arr = s.split(",");

Common Mistakes

✔ Using == instead of equals()
✔ Forgetting immutability of String
✔ Ignoring index bounds in charAt()

Real‑Time Use Cases

String methods are used in:
✔ Form validation
✔ Searching data
✔ Parsing input
✔ Web and API responses
