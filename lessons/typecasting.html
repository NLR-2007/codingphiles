Type Casting in Java

Type casting is the process of converting one data type into another.

Types of Type Casting

✔ Implicit Type Casting (Widening)
✔ Explicit Type Casting (Narrowing)

Implicit Type Casting

Automatic conversion of smaller data type to larger data type.

Example:
int a = 10;
double b = a;

Explicit Type Casting

Manual conversion of larger data type to smaller data type.

Example:
double x = 10.5;
int y = (int)x;

Type Casting (Exam Ready)

Type casting converts one data type into another using implicit or explicit methods.
