Install JDK (Java Development Kit)

To write, compile, and run Java programs, you must install the Java Development Kit (JDK) on your system.

JDK provides all the tools required for Java development.

What is JDK?

JDK stands for Java Development Kit.

It is a software package that includes:
✔ JRE (Java Runtime Environment)
✔ Java Compiler (javac)
✔ Java Development tools

Without JDK, Java programs cannot be compiled.

Why Do We Need to Install JDK?

✔ To write Java programs
✔ To compile Java source code
✔ To run Java applications
✔ To use Java development tools

System Requirements

Before installing JDK, make sure your system has:
✔ Windows / Linux / macOS
✔ Minimum 4 GB RAM (recommended)
✔ Internet connection

Steps to Install JDK on Windows

Step 1: Open your web browser
Step 2: Go to the official Java website
Step 3: Download the latest JDK for Windows
Step 4: Run the installer (.exe file)
Step 5: Follow the installation wizard
Step 6: Click Finish after installation

Steps to Install JDK on Linux

Step 1: Open Terminal
Step 2: Update package list
Step 3: Install OpenJDK package
Step 4: Verify installation

Example (Ubuntu):
sudo apt update
sudo apt install openjdk-17-jdk

Steps to Install JDK on macOS

Step 1: Download JDK for macOS
Step 2: Open the .dmg file
Step 3: Run the installer
Step 4: Follow on-screen instructions

Set JAVA_HOME Environment Variable

After installing JDK, you must set the JAVA_HOME path.

Set JAVA_HOME on Windows

Step 1: Open Control Panel
Step 2: Go to System → Advanced System Settings
Step 3: Click Environment Variables
Step 4: Add JAVA_HOME variable
Step 5: Set value to JDK installation path

Example:
C:\Program Files\Java\jdk-17

Set JAVA_HOME on Linux / macOS

Add the following line to .bashrc or .zshrc file:

export JAVA_HOME=/usr/lib/jvm/java-17-openjdk
export PATH=$JAVA_HOME/bin:$PATH

Verify JDK Installation

Open Command Prompt or Terminal and type:

java -version
javac -version

If version information appears, JDK is installed successfully.

Common Installation Issues

✔ JAVA_HOME not set correctly
✔ Path variable missing
✔ Multiple Java versions installed

Tips After Installing JDK

✔ Restart system after installation
✔ Use latest LTS version (Java 17 recommended)
✔ Install an IDE (Eclipse, IntelliJ, VS Code)
