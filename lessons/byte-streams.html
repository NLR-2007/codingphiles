Byte Streams in Java

Byte Streams in Java are used to perform input and output operations on binary data such as images, audio, video, and files.

They work with 8-bit bytes.

What are Streams?

A stream is a sequence of data.

✔ Input Stream → Reads data
✔ Output Stream → Writes data

Types of Streams in Java

Java provides two main types of streams:

✔ Byte Streams
✔ Character Streams

Byte Streams → Binary data
Character Streams → Text data

Byte Stream Classes

Byte Streams are divided into:

✔ InputStream (abstract class)
✔ OutputStream (abstract class)

InputStream

InputStream is used to read byte data from a source.

Common subclasses:

✔ FileInputStream
✔ BufferedInputStream
✔ DataInputStream

OutputStream

OutputStream is used to write byte data to a destination.

Common subclasses:

✔ FileOutputStream
✔ BufferedOutputStream
✔ DataOutputStream

FileInputStream

Used to read data from a file byte by byte.

Example

import java.io.*;

class Demo {
  public static void main(String[] args) throws Exception {
    FileInputStream fis = new FileInputStream("data.txt");
    int i;
    while ((i = fis.read()) != -1) {
      System.out.print((char) i);
    }
    fis.close();
  }
}

FileOutputStream

Used to write byte data to a file.

Example

import java.io.*;

class Demo {
  public static void main(String[] args) throws Exception {
    FileOutputStream fos = new FileOutputStream("data.txt");
    String msg = "Hello Java";
    byte[] b = msg.getBytes();
    fos.write(b);
    fos.close();
  }
}

Buffered Byte Streams

Buffered streams improve performance by reducing disk access.

✔ BufferedInputStream
✔ BufferedOutputStream

BufferedInputStream Example

BufferedInputStream bis = new BufferedInputStream(new FileInputStream("data.txt"));

Advantages of Byte Streams

✔ Can handle all file types
✔ Suitable for binary data
✔ Low-level I/O operations

Limitations

✔ Not suitable for text processing
✔ Character encoding not handled

Byte Streams vs Character Streams

✔ Byte Streams → Binary data
✔ Character Streams → Text data

✔ Byte Streams → 8-bit
✔ Character Streams → 16-bit Unicode

Real-Time Examples

✔ Image file reading/writing
✔ Audio and video files
✔ File copy operations
✔ Network data transfer

Common Mistakes

✔ Forgetting to close streams
✔ Using byte streams for text data
✔ Ignoring buffer for large files

Byte Streams (Interview / Exam Ready)

Byte streams in Java are used to perform input and output operations on binary data. They are based on InputStream and OutputStream classes and are suitable for handling files such as images, audio, and videos. Common classes include FileInputStream and FileOutputStream.
