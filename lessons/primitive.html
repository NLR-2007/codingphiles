Primitive Data Types

Primitive data types store simple values and are predefined in Java.

List of Primitive Data Types

✔ byte – 1 byte
✔ short – 2 bytes
✔ int – 4 bytes
✔ long – 8 bytes
✔ float – 4 bytes
✔ double – 8 bytes
✔ char – 2 bytes
✔ boolean – 1 bit

Example

int age = 25;
float price = 99.5f;
char grade = 'A';
boolean status = true;
