public Access Modifier in Java

The public access modifier allows class members to be accessed from anywhere.

It is the least restrictive access modifier in Java.

What is public?

When a class member is declared as public, it is accessible:

✔ Within the same class
✔ Within the same package
✔ From subclasses (same or different package)
✔ From any other class

Where Can public Be Used?

The public access modifier can be applied to:

✔ Classes
✔ Variables
✔ Methods
✔ Constructors

public Class

A public class can be accessed from anywhere in the application.

Example

public class Test {
  public void show() {
    System.out.println("Public Class");
  }
}

✔ File name must match public class name

public Variables

A public variable can be accessed directly from any class.

Example

class Student {
  public int id;
}

Student s = new Student();
s.id = 10;

⚠ Not recommended due to security issues

public Methods

A public method can be called from anywhere.

Example

class Demo {
  public void display() {
    System.out.println("Public Method");
  }
}

public Constructor

A public constructor allows object creation from anywhere.

Example

class Sample {
  public Sample() {
    System.out.println("Object Created");
  }
}

public and Packages

Public members are accessible across packages.

✔ Class must be public
✔ Member must be public

When to Use public?

Use public when:

✔ Creating APIs
✔ Writing library methods
✔ Defining entry points (main method)
✔ Exposing functionality to other packages

public vs Other Access Modifiers

✔ private → Same class only
✔ default → Same package only
✔ protected → Same package + subclass
✔ public → Anywhere

Advantages of public

✔ Maximum accessibility
✔ Supports modular programming
✔ Essential for reusable libraries

Disadvantages of public

✔ Less security
✔ Breaks encapsulation if misused
✔ Can expose internal implementation

Real-Time Examples

✔ main() method
✔ Public APIs (Collections, Math)
✔ Framework service methods

Common Mistakes

✔ Making variables public instead of methods
✔ Overusing public access
✔ Ignoring encapsulation principles

public Access Modifier (Interview / Exam Ready)

The public access modifier in Java allows access to classes, methods, variables, and constructors from anywhere. It is the least restrictive modifier and is commonly used for APIs, libraries, and entry points, but should be used carefully to maintain encapsulation and security.
