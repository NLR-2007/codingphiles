Bounded Types in Java

Bounded Types restrict the types that can be used as type arguments.

They are used to enforce constraints on generic types.

Upper Bounded Type

Syntax:

<T extends Number>

Example:

class Test<T extends Number> {
  T value;
}

Why Bounded Types?

✔ Allow only specific class types
✔ Enable use of specific methods
✔ Improve type safety

Multiple Bounds

Syntax:

<T extends Class & Interface>
