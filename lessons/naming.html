Java Naming Conventions

Naming conventions define how identifiers should be named in Java programs.

Common Naming Rules

✔ Class names → PascalCase
✔ Method names → camelCase
✔ Variable names → camelCase
✔ Constants → UPPER_CASE

Examples

class StudentDetails
int studentAge
final int MAX_LIMIT
