Wildcards in Java Generics

Wildcards are represented using the ? symbol.

They allow flexibility when working with unknown generic types.

Types of Wildcards

✔ Unbounded Wildcard (?)
✔ Upper Bounded Wildcard (? extends T)
✔ Lower Bounded Wildcard (? super T)

Unbounded Wildcard

Example:

void display(List<?> list) {
  // any type
}

Upper Bounded Wildcard

Example:

void sum(List<? extends Number> list) {
  // Number or subclass
}

Lower Bounded Wildcard

Example:

void add(List<? super Integer> list) {
  // Integer or superclass
}

Advantages

✔ Greater flexibility
✔ Better API design
✔ Works well with collections
