Nested if Statement

A nested if statement means an if statement inside another if statement.

Why Nested if?

Used when multiple conditions must be checked in sequence.

Example

int age = 22;
int weight = 55;
if(age >= 18) {
  if(weight >= 50) {
    System.out.println("Eligible");
  }
}

Nested if (Exam Ready)

Nested if statements are used to test multiple dependent conditions.
