Method Area in Java

Method Area stores class-level information.

What is Stored in Method Area?

✔ Class structure
✔ Method code
✔ Static variables
✔ Constant pool

Example

class Test {
  static int x = 10;
}

✔ static variable x → Method Area

Method Area Error

If memory is full, OutOfMemoryError: Metaspace occurs.

Method Area (Easy to Remember)

Method Area stores class data, not object data.
