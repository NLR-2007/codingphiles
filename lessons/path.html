Path & Classpath in Java

Path and Classpath are environment variables used to locate Java tools and class files.

What is Path?

Path tells the operating system where Java tools like javac and java are located.

Without setting Path, Java commands will not work from any directory.

Setting Path

✔ Add JDK bin directory to system Path
✔ Required to run javac and java commands

What is Classpath?

Classpath tells JVM where to find user‑defined classes and libraries during execution.

Difference Between Path and Classpath

Path: Locates Java tools
Classpath: Locates Java classes

Path & Classpath (Exam Ready)

Path is used to locate Java tools, while Classpath is used to locate class files during program execution.
