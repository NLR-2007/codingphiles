Literals in Java

Literals are constant values assigned to variables.

Types of Literals

✔ Integer literals
✔ Floating‑point literals
✔ Character literals
✔ String literals
✔ Boolean literals

Examples

int a = 10;
float b = 5.5f;
char c = 'A';
String s = "Java";
boolean flag = true;
