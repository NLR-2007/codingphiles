String Class in Java

The String class in Java is used to create and manipulate a sequence of characters.

It is one of the most important and frequently used classes in Java programming.

Package of String Class

The String class belongs to the java.lang package.

Because of this, it is automatically imported into every Java program.

String is an Object

In Java, String is not a primitive type. It is a class, and every string you create is an object.

Example:

String name = "Java";

How String Objects are Stored

Java stores string objects in two memory areas:

✔ String Constant Pool (SCP)
✔ Heap Memory

String Constant Pool

When a string literal is created, Java checks the SCP first. If the value already exists, the reference is reused.

This improves memory efficiency.

Heap Memory

When a string is created using the new keyword, a new object is always created in heap memory.

Example

String s1 = "Java";
String s2 = "Java";
String s3 = new String("Java");

✔ s1 and s2 refer to the same object
✔ s3 refers to a different object

Immutability of String Class

Strings in Java are immutable. Once a string object is created, its value cannot be changed.

Example:

String s = "Java";
s.concat(" Programming");

The original string remains unchanged.

Why String Class is Immutable?

✔ Security (used in passwords, URLs)
✔ Thread safety
✔ Performance optimization using SCP

Commonly Used String Class Methods

✔ length()
✔ charAt()
✔ substring()
✔ equals()
✔ equalsIgnoreCase()
✔ compareTo()
✔ toUpperCase()
✔ toLowerCase()
✔ trim()
✔ replace()

Example Using String Methods

String s = " Java Programming ";
s.length();
s.trim();
s.toUpperCase();
s.charAt(0);

String Comparison

String objects can be compared using:

✔ == operator (reference comparison)
✔ equals() method (content comparison)

Example

String a = "Java";
String b = "Java";
a == b; // true
a.equals(b); // true

Advantages of String Class

✔ Easy text handling
✔ Secure and reliable
✔ Rich built-in methods
✔ Memory efficient

Limitations of String Class

✔ Immutable (not suitable for frequent modifications)
✔ Performance issues with heavy concatenation

String vs StringBuffer vs StringBuilder

✔ String → Immutable
✔ StringBuffer → Mutable & Thread-safe
✔ StringBuilder → Mutable & Faster

Real-Time Use Cases of String Class

The String class is used in:

✔ User input handling
✔ Login and authentication systems
✔ File paths and URLs
✔ Database queries
✔ Web and API responses

String Class (Interview / Exam Ready)

The String class in Java represents an immutable sequence of characters, stored in the String Constant Pool or heap memory, and provides a rich set of methods to manipulate and compare text data.
