Java Comments

Comments are used to explain code and are ignored by the compiler.

Types of Comments

✔ Single-line comment (//)
✔ Multi-line comment (/* */)
✔ Documentation comment (/** */)

Examples

// This is a comment
/* Multi-line comment */
/** Documentation comment */

Why Comments are Important

✔ Improves readability
✔ Helps maintenance
✔ Useful for documentation

Java Comments (Exam Ready)

Java comments explain code logic and do not affect program execution.
