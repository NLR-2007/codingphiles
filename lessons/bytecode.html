Bytecode in Java

Bytecode is an intermediate machine-independent code generated by the Java compiler.

Bytecode is stored in .class files.

Why Bytecode?

✔ Platform Independent
✔ Secure Execution
✔ Optimized by JVM

Example

javac Hello.java → Hello.class

✔ Hello.class contains bytecode

Interview Ready

Bytecode allows Java programs to run on any operating system using JVM.
