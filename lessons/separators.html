Separators in Java

Separators are symbols used to separate code elements.

Common Separators

✔ ( ) – Method calls
✔ { } – Blocks
✔ [ ] – Arrays
✔ ; – Statement termination
✔ , – Variable separation

Example

int a, b;
if(a > b) { }
int arr[] = new int[5];
