Operators in Java

Operators are symbols used to perform operations on variables and values.

Types of Operators

✔ Arithmetic (+, -, *, /)
✔ Relational (>, <, ==)
✔ Logical (&&, ||, !)
✔ Assignment (=, +=)
✔ Unary (++ , --)

Example

int a = 10, b = 5;
int sum = a + b;

Operators (Exam Ready)

Operators perform operations on operands to produce a result.
