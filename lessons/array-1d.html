Single Dimensional Array in Java

A Single Dimensional Array is a collection of elements of the same data type stored in a linear sequence.

Each element in the array is accessed using an index value. Indexing in Java arrays always starts from 0.

Why do we need Arrays?

Arrays are used when we need to store multiple values of the same type using a single variable.

Without arrays:
int a1, a2, a3, a4, a5;

With arrays:
int[] a = new int[5];

Declaration of Single Dimensional Array

Syntax:

dataType[] arrayName;

Example:
int[] numbers;

Array Creation

Syntax:

arrayName = new dataType[size];

Example:
numbers = new int[5];

Declaration and Creation in One Line

int[] numbers = new int[5];

Array Initialization

Array elements can be initialized in two ways.

1. Using Index

int[] marks = new int[3];
marks[0] = 80;
marks[1] = 85;
marks[2] = 90;

2. Using Array Literal

int[] marks = {80, 85, 90};

Accessing Array Elements

Array elements are accessed using index numbers.

Example:
System.out.println(marks[0]);

Array Index Rules

✔ Index starts from 0
✔ Last index is (length - 1)
✔ Accessing invalid index causes runtime error

Array Length

Java provides a built-in length property to find the size of an array.

Example:
int size = marks.length;

Traversing a Single Dimensional Array

Traversing means accessing all elements one by one.

Using for Loop

for(int i = 0; i < marks.length; i++) {
  System.out.println(marks[i]);
}

Using Enhanced for Loop

for(int m : marks) {
  System.out.println(m);
}

Advantages of Single Dimensional Array

✔ Stores multiple values efficiently
✔ Easy access using index
✔ Improves code readability

Limitations of Single Dimensional Array

✔ Fixed size
✔ Stores only same data type
✔ Cannot grow dynamically

Real-Time Use Cases

Single dimensional arrays are used in:
✔ Storing student marks
✔ Managing employee IDs
✔ Holding sensor values
✔ Processing lists of numbers

Common Mistakes

✔ ArrayIndexOutOfBoundsException
✔ Forgetting array size
✔ Accessing uninitialized elements

Single Dimensional Array (Interview / Exam Ready)
