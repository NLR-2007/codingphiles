Collectors in Java

Collectors are used to collect stream elements into a result such as a List, Set, or Map.

Common Collectors

✔ Collectors.toList()
✔ Collectors.toSet()
✔ Collectors.toMap()
✔ Collectors.groupingBy()

Example

List result = list.stream()
  .filter(n -> n > 10)
  .collect(Collectors.toList());
