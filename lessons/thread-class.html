Thread Class in Java

The Thread class in Java is used to create and control threads.

It is present in the java.lang package and provides built‑in support for multithreading.

What is Thread Class?

The Thread class represents a thread of execution in a Java program.

✔ Each thread is an instance of Thread class
✔ JVM schedules threads for execution

Why Thread Class?

Thread class is used to:

✔ Create threads
✔ Start thread execution
✔ Control thread behavior
✔ Manage thread life cycle

Creating Thread Using Thread Class

Steps:

✔ Extend Thread class
✔ Override run() method
✔ Create object of class
✔ Call start()

Example: Extending Thread Class

class MyThread extends Thread {
  public void run() {
    System.out.println("Thread is running");
  }
}

class Test {
  public static void main(String[] args) {
    MyThread t = new MyThread();
    t.start();
  }
}

start() vs run()

✔ start() → Creates new thread and executes run()
✔ run() → Executes like normal method

⚠ Always use start() to begin a new thread

Important Methods of Thread Class

Commonly used Thread class methods:

✔ start()
✔ run()
✔ sleep()
✔ join()
✔ getName()
✔ setName()
✔ getPriority()
✔ setPriority()
✔ isAlive()
✔ currentThread()

Thread sleep()

Pauses execution of current thread for specified time.

Thread.sleep(1000); // 1 second

Thread join()

Makes one thread wait for another thread to finish execution.

Thread Priority

Thread priority decides execution preference.

✔ MIN_PRIORITY → 1
✔ NORM_PRIORITY → 5
✔ MAX_PRIORITY → 10

Thread currentThread()

Returns reference to currently executing thread.

Thread t = Thread.currentThread();

Thread isAlive()

Checks whether a thread is alive.

✔ true → thread is running or ready
✔ false → thread is terminated

Thread Class vs Runnable Interface

✔ Thread → Inherits Thread class
✔ Runnable → Implements Runnable interface

✔ Thread → Cannot extend another class
✔ Runnable → Can extend another class

✔ Runnable → Preferred approach

Limitations of Thread Class

✔ Java does not support multiple inheritance
✔ Extending Thread blocks extending other classes

Real‑Time Examples

✔ File download threads
✔ Game animation threads
✔ Background services
✔ Server request handling

Common Mistakes

✔ Calling run() instead of start()
✔ Calling start() twice
✔ Ignoring thread synchronization

Thread Class (Interview / Exam Ready)

The Thread class in Java is used to create and manage threads. A thread can be created by extending the Thread class and overriding the run() method. The start() method is used to begin execution, and the Thread class provides several methods to control thread behavior and life cycle.
