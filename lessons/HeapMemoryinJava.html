Heap Memory in Java

Heap Memory is a runtime memory area in Java used to store objects and instance variables.

Heap memory is created when the JVM starts and is shared among all threads.

Why do we need Heap Memory?

Heap memory is required to store objects dynamically so they can be accessed throughout the program.
