this Keyword in Java

The this keyword in Java is a reference variable that refers to the current object.

It is commonly used inside instance methods and constructors.

Why Do We Need the this Keyword?

The this keyword is mainly used to:
✔ Differentiate instance variables from local variables
✔ Refer to the current class object
✔ Call another constructor of the same class
✔ Pass the current object as an argument

1. this to Differentiate Instance and Local Variables

When instance variables and method/constructor parameters have the same name, this is used to avoid confusion.

Example

class Student {
  int id;

  Student(int id) {
    this.id = id;
  }
}

Explanation:
✔ this.id → instance variable
✔ id → constructor parameter

2. this to Refer Current Class Object

The this keyword always points to the object that is currently calling the method.

Example

class Demo {
  void show() {
    System.out.println(this);
  }
}

Here, this refers to the current object.

3. this to Call Another Constructor (Constructor Chaining)

The this() statement is used to call another constructor of the same class.

It must be the first statement inside the constructor.

Example

class Demo {
  Demo() {
    this(10);
    System.out.println("Default Constructor");
  }

  Demo(int x) {
    System.out.println(x);
  }
}

4. this to Pass Current Object as Method Argument

The current object can be passed to another method using this.

Example

class Test {
  void display(Test t) { }

  void show() {
    display(this);
  }
}

5. this to Return Current Object

The this keyword can also be used to return the current object.

Example

class Demo {
  Demo getObject() {
    return this;
  }
}

Important Rules of this Keyword

✔ this cannot be used in static context
✔ this refers only to instance members
✔ this() must be the first statement in constructor

Common Mistakes

✔ Using this in static methods
✔ Forgetting this when variable names are same
✔ Calling this() not as first statement

Real-Time Example

✔ BankAccount → this.balance = balance
✔ Student → this.id = id
✔ Employee → this.salary = salary

this Keyword (Interview / Exam Ready)

The this keyword in Java is a reference to the current object. It is used to differentiate instance variables from local variables, call constructors of the same class, and pass or return the current object.
