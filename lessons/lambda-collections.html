Lambda with Collections

Lambda expressions simplify operations on collections such as List, Set, and Map.

Using forEach()

The forEach() method is commonly used with lambda expressions.

Example with List

List list = Arrays.asList(10, 20, 30);
list.forEach(n -> System.out.println(n));

Example with Map

Map map = new HashMap();
map.forEach((k, v) -> System.out.println(k + " " + v));
