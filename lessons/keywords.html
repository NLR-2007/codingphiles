Java Keywords

Keywords are reserved words in Java that have a predefined meaning.

They cannot be used as variable, method, or class names.

Examples of Java Keywords

✔ class
✔ public
✔ static
✔ void
✔ if, else
✔ for, while
✔ return

Important Points

✔ Java keywords are case‑sensitive
✔ Total keywords ≈ 50+ (version dependent)
