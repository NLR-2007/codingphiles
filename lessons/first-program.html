First Java Program

The first Java program helps beginners understand Java syntax and execution flow.

Hello World Program

class HelloWorld {
  public static void main(String[] args) {
    System.out.println("Hello World");
  }
}

Explanation

✔ class HelloWorld: Defines the class
✔ main(): Execution starts here
✔ System.out.println(): Prints output

Why Hello World?

✔ Confirms Java installation
✔ Verifies compilation & execution
✔ Introduces basic syntax
