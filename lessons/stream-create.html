Stream Creation in Java

A Stream represents a sequence of elements that supports functional-style operations.

Ways to Create Streams

✔ From Collections
✔ From Arrays
✔ Using Stream.of()

Examples

list.stream();
Arrays.stream(arr);
Stream.of(1, 2, 3);

Key Points

✔ Streams do not store data
✔ They operate on data source
✔ Introduced in Java 8
