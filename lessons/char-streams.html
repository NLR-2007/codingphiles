Character Streams in Java

Character Streams are used to perform input and output operations on text data.

They work with 16‑bit Unicode characters.

Why Character Streams?

✔ Suitable for text files
✔ Handles character encoding automatically
✔ More efficient for text processing

Character Stream Classes

✔ Reader (abstract class)
✔ Writer (abstract class)

Examples

✔ FileReader
✔ FileWriter
