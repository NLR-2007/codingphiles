Java Source File Rules

Java source files must follow certain rules to compile and execute successfully.

Important Rules

✔ File extension must be .java
✔ Public class name must match file name
✔ Only one public class per file
✔ main() must be inside a class

Examples

✔ Correct: Hello.java → public class Hello
✖ Incorrect: Test.java → public class Demo

Why Rules Matter

Breaking source file rules causes compilation errors.
