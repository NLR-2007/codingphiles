Array Initialization in Java

Array Initialization is the process of assigning values to the elements of an array.

In Java, arrays can be initialized in multiple ways depending on the requirement.

Why is Array Initialization Important?

Without initialization, array elements contain default values, which may lead to incorrect results.

Array initialization is used to:
✔ Store actual data
✔ Perform calculations
✔ Process meaningful values

Types of Array Initialization

Java supports the following ways to initialize arrays:
✔ Initialization using index
✔ Initialization using array literal
✔ Initialization using loop

1. Initialization Using Index

In this method, array elements are assigned values one by one using index positions.

Example:

int[] marks = new int[3];
marks[0] = 80;
marks[1] = 85;
marks[2] = 90;

2. Initialization Using Array Literal

Array literals allow initializing the array at the time of declaration.

Example:

int[] marks = {80, 85, 90};

This method is simple and commonly used in programs.

3. Initialization Using new Keyword

Arrays can also be initialized directly using the new keyword.

Example:

int[] numbers = new int[] {1, 2, 3, 4, 5};

Initialization of 2D Arrays

Two dimensional arrays can also be initialized in multiple ways.

Using Index Values

int[][] matrix = new int[2][2];
matrix[0][0] = 1;
matrix[0][1] = 2;
matrix[1][0] = 3;
matrix[1][1] = 4;

Using Array Literal

int[][] matrix = {
  {1, 2},
  {3, 4}
};

Default Values in Array

If an array is declared but not initialized, Java assigns default values.

Default values:
✔ int → 0
✔ float → 0.0
✔ boolean → false
✔ char → '\\u0000'
✔ Object → null

Initialization Using Loop

Arrays can be initialized dynamically using loops.

Example:

int[] arr = new int[5];
for(int i = 0; i < arr.length; i++) {
  arr[i] = i * 10;
}

Common Mistakes

✔ Accessing array before initialization
✔ Using wrong index value
✔ Forgetting array size

Real‑Time Use Cases

Array initialization is used in:
✔ Storing marks
✔ Loading configuration values
✔ Initializing test data
✔ Creating lookup tables
