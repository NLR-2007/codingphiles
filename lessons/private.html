private Access Modifier in Java

The private access modifier is used to restrict access to class members.

A private member is accessible only within the same class.

What is private?

The private keyword is the most restrictive access modifier in Java.

It is mainly used for:
✔ Data hiding
✔ Security
✔ Encapsulation

Where Can private Be Used?

The private access modifier can be applied to:

✔ Variables
✔ Methods
✔ Constructors

Note:
Classes themselves cannot be private.

private Variables

When a variable is declared as private, it cannot be accessed directly outside the class.

Example

class Student {
  private int age;
}

✔ age is hidden from outside access

Accessing private Variables

Private variables are accessed using public getter and setter methods.

Example

class Student {
  private int age;

  public void setAge(int age) {
    if(age > 0) {
      this.age = age;
    }
  }

  public int getAge() {
    return age;
  }
}

private Methods

A private method can be accessed only within the same class.

Example

class Demo {
  private void show() {
    System.out.println("Private Method");
  }

  void call() {
    show();
  }
}

✔ show() cannot be called from another class

private Constructor

A private constructor prevents object creation from outside the class.

Example

class Singleton {
  private Singleton() { }
}

Used in:
✔ Singleton Design Pattern
✔ Utility classes

private and Inheritance

Private members are not inherited by child classes.

✔ Child class cannot access private members
✔ They belong strictly to the parent class

private vs Other Access Modifiers

✔ private → Same class only
✔ default → Same package
✔ protected → Same package + subclass
✔ public → Anywhere

Advantages of private

✔ Strong data security
✔ Supports encapsulation
✔ Prevents unauthorized access
✔ Improves maintainability

Real-Time Examples

✔ BankAccount → balance is private
✔ ATM → PIN is private
✔ Employee → salary is private

Common Mistakes

✔ Trying to access private members outside class
✔ Forgetting getters and setters
✔ Assuming private members are inherited

private (Interview / Exam Ready)

The private access modifier in Java restricts access to class members so that they are accessible only within the same class. It is mainly used for data hiding and supports encapsulation by preventing direct access to internal variables and methods.
