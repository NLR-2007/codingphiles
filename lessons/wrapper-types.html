Integer Wrapper Class

Integer is a wrapper class for the primitive type int.

Example

Integer i = Integer.valueOf(100);
int a = i.intValue();

Common Methods

✔ parseInt()
✔ valueOf()
✔ toString()

Double Wrapper Class

Double is a wrapper class for the primitive type double.

Example

Double d = 10.5;
double x = d;

Character Wrapper Class

Character is a wrapper class for the primitive type char.

Example

Character c = 'A';
char ch = c;

Useful Methods

✔ isDigit()
✔ isLetter()
✔ isUpperCase()

Boolean Wrapper Class

Boolean is a wrapper class for the primitive type boolean.

Example

Boolean b = true;
boolean flag = b;
