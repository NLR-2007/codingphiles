Java Directory Structure

Java follows a standard directory structure to organize source files, class files, and packages.

Common Java Project Structure

✔ src – Source code (.java files)
✔ bin / out – Compiled files (.class files)
✔ lib – External libraries
✔ packages – Organized code structure

Why Directory Structure is Important

✔ Better code organization
✔ Easy maintenance
✔ Supports package concept
