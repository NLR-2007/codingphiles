Identifiers in Java

Identifiers are names used to identify variables, methods, classes, and packages.

Rules for Identifiers

✔ Can contain letters, digits, _ and $
✔ Cannot start with a digit
✔ Cannot be a keyword
✔ Case‑sensitive

Examples

✔ valid: totalMarks, student_age
✖ invalid: 1value, class
