Anonymous Array in Java

An Anonymous Array in Java is an array that is created without assigning it to a variable.

It is mainly used when an array is required only once, usually at the time of method calling.

Why is it called Anonymous Array?

It is called anonymous because:
✔ No reference variable is used
✔ The array has no name
✔ It is used directly

Syntax of Anonymous Array

new dataType[] { value1, value2, value3, ... }

Example of Anonymous Array

System.out.println(new int[] {10, 20, 30}[1]);

Output:
20

Anonymous Array with Method

Anonymous arrays are most commonly used while passing data to methods.

Example

class Test {
  static void printArray(int[] arr) {
    for(int i : arr) {
      System.out.println(i);
    }
  }

  public static void main(String[] args) {
    printArray(new int[] {5, 10, 15});
  }
}

Explanation

✔ Array is created at runtime
✔ Passed directly to the method
✔ No variable is declared

Anonymous Array in 2D

Anonymous arrays can also be used with two dimensional arrays.

Example:

printMatrix(new int[][] {
  {1, 2},
  {3, 4}
});

When to Use Anonymous Arrays?

Anonymous arrays are useful when:
✔ Array is used only once
✔ Temporary data is needed
✔ Clean and short code is preferred

Advantages of Anonymous Array

✔ No need to declare a variable
✔ Saves memory for short‑term usage
✔ Makes code concise

Limitations of Anonymous Array

✔ Cannot reuse the array
✔ Hard to debug for beginners
✔ Not suitable for complex logic

Real‑Time Use Cases

Anonymous arrays are used in:
✔ Method calls with fixed values
✔ Test cases
✔ One‑time calculations

Common Mistakes

✔ Trying to reuse anonymous array
✔ Using it for large datasets
✔ Overusing and reducing readability

Anonymous Array (Interview / Exam Ready)
