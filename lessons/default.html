default Access Modifier in Java

The default access modifier is applied when no access modifier is specified.

It is also called package‑private.

What is default Access?

When no keyword (private, protected, public) is used, Java applies default access.

Members with default access are accessible only within the same package.

Where Can default Be Used?

The default access modifier can be applied to:

✔ Variables
✔ Methods
✔ Classes
✔ Constructors

default Variables

A variable with default access can be accessed only by classes in the same package.

Example

class Student {
  int id; // default access
}

✔ Accessible within same package
✖ Not accessible outside the package

default Methods

A method with default access can be called only within the same package.

Example

class Demo {
  void show() {
    System.out.println("Default Method");
  }
}

default Class

A class without an access modifier has default access.

Such a class:
✔ Can be used within the same package
✖ Cannot be accessed from another package

Example

class Test { } // default class

default Constructor

A constructor without an access modifier has default access.

It can be called only within the same package.

default and Packages

Default access is package‑based.

✔ Same package → Access allowed
✖ Different package → Access denied

Even subclasses in other packages cannot access default members.

default vs Other Access Modifiers

✔ private → Same class only
✔ default → Same package only
✔ protected → Same package + subclass
✔ public → Anywhere

Advantages of default Access

✔ Prevents unnecessary exposure
✔ Useful for internal package logic
✔ Supports modular design

Real-Time Examples

✔ Utility classes used only inside package
✔ Helper methods for internal logic
✔ Package-level service classes

Common Mistakes

✔ Assuming default is public
✔ Trying to access default members from another package
✔ Confusing default with protected

default Access Modifier (Interview / Exam Ready)

The default access modifier in Java is applied when no access modifier is specified. It allows access to class members only within the same package and is also known as package‑private access.
