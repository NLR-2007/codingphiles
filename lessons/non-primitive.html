Non‑Primitive Data Types

Non‑primitive data types store complex data and references.

Types of Non‑Primitive Data Types

✔ String
✔ Array
✔ Class
✔ Object
✔ Interface

Example

String name = "Java";
int[] marks = {80, 90, 85};

Difference from Primitive

✔ Can store multiple values
✔ Have methods
✔ Store references

Non‑Primitive Data Types (Exam Ready)

Non‑primitive data types store references to objects and complex data.
