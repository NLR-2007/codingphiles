if Statement in Java

The if statement is a decision‑making statement in Java. It is used to execute a block of code only when a given condition is true.

If the condition evaluates to false, the code inside the if block is skipped.

Why do we use if Statement?

In real‑world applications, decisions are required.

Examples:
✔ Check if a user is eligible to vote
✔ Check if marks are pass or fail
✔ Check if balance is sufficient

The if statement helps Java programs make such decisions.

Syntax of if Statement

if(condition) {
  // code to execute if condition is true
}

The condition must return a boolean value (true or false).

Flow of Execution

✔ Condition is evaluated
✔ If condition is true → code inside if executes
✔ If condition is false → code is skipped

Simple Example

int age = 20;
if(age >= 18) {
  System.out.println("Eligible to vote");
}

Output:
Eligible to vote

Explanation of Example

✔ age variable stores value 20
✔ Condition checks if age is greater than or equal to 18
✔ Condition is true, so the message is printed

if Statement with Relational Operators

The if statement is commonly used with relational operators.

Example:

int a = 10;
int b = 5;
if(a > b) {
  System.out.println("a is greater than b");
}

if Statement with Logical Operators

Logical operators can be used to check multiple conditions.

Example:

int age = 25;
if(age > 18 && age < 60) {
  System.out.println("Working age group");
}

Important Rules of if Statement

✔ Condition must be boolean
✔ Curly braces are optional for single statement
✔ Best practice is to always use braces

if Statement without Curly Braces

Example:

int x = 5;
if(x > 0)
  System.out.println("Positive number");

Note: Only one statement is controlled by the if condition.

Common Mistakes

✔ Using = instead of ==
✔ Missing braces for multiple statements
✔ Using non‑boolean conditions

Real‑Time Use Case

if statement is used in:
✔ Login validation
✔ Banking applications
✔ Form validations
✔ Game logic

Advantages of if Statement

✔ Simple decision making
✔ Easy to understand
✔ Improves program logic

Limitations of if Statement

✔ Handles only one condition
✔ For multiple conditions, else or switch is required
