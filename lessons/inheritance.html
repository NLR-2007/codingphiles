Inheritance in Java

Inheritance is one of the core Object-Oriented Programming (OOP) concepts.

It allows one class to acquire the properties and methods of another class.

What is Inheritance?

Inheritance represents an IS-A relationship.

It enables:
✔ Code reusability
✔ Method overriding
✔ Runtime polymorphism

Basic Syntax of Inheritance

class Parent {
  // parent members
}

class Child extends Parent {
  // child members
}

Here:
✔ Parent → Superclass
✔ Child → Subclass

Example of Inheritance

class Animal {
  void eat() {
    System.out.println("Eating");
  }
}

class Dog extends Animal {
  void bark() {
    System.out.println("Barking");
  }
}

Dog d = new Dog();
d.eat();
d.bark();

✔ Dog inherits eat() from Animal

Types of Inheritance in Java

Java supports the following types using classes:

✔ Single Inheritance
✔ Multilevel Inheritance
✔ Hierarchical Inheritance

Java does not support multiple inheritance using classes (to avoid ambiguity).

1. Single Inheritance

One child class inherits from one parent class.

class A { }
class B extends A { }

2. Multilevel Inheritance

A class inherits from a child class.

class A { }
class B extends A { }
class C extends B { }

3. Hierarchical Inheritance

Multiple child classes inherit from one parent class.

class A { }
class B extends A { }
class C extends A { }

Why Java Does Not Support Multiple Inheritance?

Multiple inheritance using classes can cause the Diamond Problem.

Java avoids ambiguity by not allowing it.

✔ Multiple inheritance is possible using interfaces

Method Overriding in Inheritance

When a child class provides its own implementation of a parent class method, it is called Method Overriding.

Rules:
✔ Method name must be same
✔ Parameters must be same
✔ IS-A relationship must exist

super Keyword (Quick Overview)

The super keyword refers to the parent class object.

Uses:
✔ Access parent class variables
✔ Call parent class methods
✔ Call parent class constructor

Advantages of Inheritance

✔ Code reusability
✔ Less duplication
✔ Easy maintenance
✔ Supports polymorphism

Real-Time Examples

✔ Employee → Manager, Developer
✔ Vehicle → Car, Bike
✔ Account → Savings, Current

Common Mistakes

✔ Forgetting IS-A relationship
✔ Using inheritance instead of composition
✔ Trying multiple inheritance with classes

Inheritance (Interview / Exam Ready)

Inheritance in Java is an OOP concept where a child class acquires the properties and methods of a parent class using the extends keyword. It promotes code reusability and supports method overriding and polymorphism.
