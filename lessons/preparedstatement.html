PreparedStatement in JDBC

PreparedStatement is used to execute parameterized SQL queries.

Advantages

✔ Prevents SQL Injection
✔ Better performance
✔ Precompiled queries

Example

PreparedStatement ps = con.prepareStatement(
  "INSERT INTO student VALUES (?, ?)"
);
ps.setInt(1, 101);
ps.setString(2, "John");
ps.executeUpdate();
