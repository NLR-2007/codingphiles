Generic Methods in Java

A Generic Method is a method that introduces its own type parameter.

It can be used in both generic and non‑generic classes.

Syntax

<T> returnType methodName(T param) {
  // code
}

Example

class Test {
  static <T> void display(T value) {
    System.out.println(value);
  }

  public static void main(String[] args) {
    display(10);
    display("Java");
  }
}

Key Points

✔ Type parameter before return type
✔ Improves method flexibility
✔ Avoids method overloading
