Array Traversal in Java

Array Traversal means accessing and processing each element of an array one by one.

Traversal is one of the most common operations performed on arrays in Java.

Why is Array Traversal Important?

Array traversal is required when we need to:

✔ Display all elements
✔ Perform calculations (sum, average, max, min)
✔ Search for an element
✔ Modify array values

Traversing a Single Dimensional Array

There are two main ways to traverse a 1D array in Java.

1. Using for Loop

The traditional for loop is commonly used when index values are required.

Example:

int[] marks = {80, 85, 90, 75};
for(int i = 0; i < marks.length; i++) {
  System.out.println(marks[i]);
}

Explanation

✔ Loop starts from index 0
✔ Continues until length - 1
✔ Each element is accessed using index

2. Using Enhanced for Loop (for-each)

The enhanced for loop is used when index values are not required.

Example:

for(int m : marks) {
  System.out.println(m);
}

Difference Between for and Enhanced for

✔ for loop → index based access
✔ enhanced for → value based access
✔ enhanced for is simpler and cleaner

Traversing a Two Dimensional Array

A 2D array requires nested loops for traversal.

Using Nested for Loop

Example:

int[][] matrix = {
  {1, 2, 3},
  {4, 5, 6}
};

for(int i = 0; i < matrix.length; i++) {
  for(int j = 0; j < matrix[i].length; j++) {
    System.out.print(matrix[i][j] + " ");
  }
  System.out.println();
}

Using while Loop for Traversal

Arrays can also be traversed using a while loop.

Example:

int i = 0;
while(i < marks.length) {
  System.out.println(marks[i]);
  i++;
}

Using do-while Loop for Traversal

The do-while loop ensures the array is traversed at least once.

Example:

int i = 0;
do {
  System.out.println(marks[i]);
  i++;
} while(i < marks.length);

Common Mistakes During Array Traversal

✔ Using <= instead of < in loop condition
✔ Forgetting array length
✔ Accessing invalid index values

Real-Time Use Cases

Array traversal is used in:

✔ Calculating total and average marks
✔ Finding maximum and minimum values
✔ Searching elements
✔ Displaying data in reports

Advantages of Proper Array Traversal

✔ Prevents runtime errors
✔ Improves program reliability
✔ Makes data processing easy

Array Traversal (Interview / Exam Ready)

Array traversal is the process of accessing each element of an array sequentially using loops such as for, while, do-while, or enhanced for loop.
