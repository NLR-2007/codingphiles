Two Dimensional Array in Java

A Two Dimensional Array (2D Array) in Java is an array of arrays.

It is used to store data in the form of rows and columns, similar to a table or matrix.

Why do we need 2D Arrays?

2D arrays are useful when data is naturally represented in tabular form.

Examples:
✔ Student marks (rows = students, columns = subjects)
✔ Matrix calculations
✔ Game boards (chess, tic‑tac‑toe)

Declaration of 2D Array

Syntax:

dataType[][] arrayName;

Example:
int[][] marks;

Creation of 2D Array

Syntax:

arrayName = new dataType[rows][columns];

Example:
marks = new int[3][3];

Declaration and Creation in One Line

int[][] marks = new int[3][3];

Initialization of 2D Array

1. Using Index Values

int[][] marks = new int[2][2];
marks[0][0] = 80;
marks[0][1] = 85;
marks[1][0] = 90;
marks[1][1] = 95;

2. Using Array Literal

int[][] marks = {
  {80, 85},
  {90, 95}
};

Accessing 2D Array Elements

Elements are accessed using row index and column index.

Example:
System.out.println(marks[1][0]);

Traversing a 2D Array

Using Nested for Loop

for(int i = 0; i < marks.length; i++) {
  for(int j = 0; j < marks[i].length; j++) {
    System.out.print(marks[i][j] + " ");
  }
  System.out.println();
}

Length in 2D Array

✔ array.length → number of rows
✔ array[row].length → number of columns

Jagged Array

A jagged array is a 2D array where each row can have a different number of columns.

Example:

int[][] arr = new int[3][];
arr[0] = new int[2];
arr[1] = new int[3];
arr[2] = new int[1];

Advantages of 2D Array

✔ Represents data in tabular form
✔ Easy to process rows and columns
✔ Useful for mathematical operations

Limitations of 2D Array

✔ Fixed size
✔ Memory wastage if not fully used
✔ Complex traversal compared to 1D array

Real‑Time Use Cases

2D arrays are used in:
✔ Student result systems
✔ Image processing
✔ Game development
✔ Matrix operations

Common Errors

✔ ArrayIndexOutOfBoundsException
✔ Incorrect loop boundaries
✔ Confusing row and column indexes

Two Dimensional Array (Interview / Exam Ready)

A two dimensional array in Java is an array of arrays used to store data in rows and columns, accessed using two index values starting from zero.
